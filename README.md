# Notes Sharing Platform

This is a simple Notes Sharing Platform built using Flask. Users can create an account, upload notes, and view or download approved notes. An admin panel is also included to manage and approve the uploaded notes.

---

## Features

- User Signup and Login
- Upload Notes
- View and Download Notes
- Search Notes
- Admin Panel
- Approve and Delete Notes
- User Dashboard
- My Notes Section

---

## Tech Stack

- Python (Flask)
- SQLite
- HTML, CSS

---

---

## How to Run

1. Clone the repository
2. Install dependencies using:
   pip install -r requirements.txt
3. Run the app:
   python app.py
4. Open browser:
   http://localhost:10000
